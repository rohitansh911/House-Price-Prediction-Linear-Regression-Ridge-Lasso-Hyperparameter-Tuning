🏠 House Price Prediction using Machine Learning
📌 Overview

This project predicts residential house prices using the Kaggle House Prices dataset.

The project demonstrates an end-to-end Machine Learning workflow including data preprocessing, exploratory data analysis, feature engineering, model building, evaluation, regularization techniques, and hyperparameter tuning.
